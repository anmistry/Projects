# How to Build:
1. Open Linux command prompt.
2. Git clone Bit Magic folder.
3. Go to Bit Magic folder.
4. Run command ```gcc -o smallsh smallsh.c``` on command line.
5. Run ```./smallsh``` on command prompt.
6. Program will execute.
